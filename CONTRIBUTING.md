
# Contribuindo

Obrigado por colaborar! Este projeto é usado para fins educacionais na disciplina **Algoritmos Avançados em Python (UNIPLAN)**.

## Fluxo de contribuição (Pull Request)

1. Faça **fork** do repositório e crie uma **branch** descritiva:
   ```bash
   git checkout -b avaliacao-final-seu-nome
   ```
2. Limite suas alterações a:
   - `arvore/tree_logic.py`
   - `grafo/graph_logic.py`
3. Confirme que o projeto roda localmente.
4. Faça commit com mensagem clara (ex.: *feat: implementa navigate_tree e connected*).
5. Abra o **Pull Request** para o repositório original.
6. No corpo do PR, inclua o **Relato de Aprendizagem** (ver README).

## Padrões de código
- Código simples, comentado e legível.
- Sem dependências externas.
- Não altere as UIs Tkinter.
